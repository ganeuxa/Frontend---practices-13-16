# 📝 Заметки (Notes PWA)

## 📖 О проекте

PWA-приложение для управления заметками с поддержкой:
- ✅ Офлайн-режима
- ✅ Push-уведомлений
- ✅ WebSocket синхронизации
- ✅ Напоминаний с функцией «Отложить»

Проект выполнен в рамках практических занятий №13–17.

---

## 🛠 Технологии

| Frontend | Backend | PWA |
|----------|---------|-----|
| HTML5, CSS3, JavaScript | Node.js, Express | Service Worker |
| Chota CSS | Socket.IO | Push API |
| LocalStorage | web-push | Manifest |

## 🚀 Быстрый старт

### 1. Установка
```bash
npm install
cd server && npm install
```

### 2. Запуск (два терминала)
```bash
# Терминал 1 - Бэкенд
cd server
node server_push.js

# Терминал 2 - Фронтенд
cd server
node server_frontend.js
```

### 3. Открыть
Перейдите на **`https://localhost:4001`**

> ⚠️ Примите самоподписанный сертификат при первом входе

---

## 🧪 Как проверить

### Заметка с напоминанием:
1. Заполните форму «Текст напоминания» + выберите дату/время
2. Нажмите **«С напоминанием»**
3. Закройте вкладку
4. Дождитесь push-уведомления

### Кнопка «Отложить»:
1. В уведомлении нажмите **«⏰ Отложить на 5 мин»**
2. Появится подтверждение
3. Через 5 минут придёт новое уведомление

### Офлайн:
1. Откройте приложение
2. Отключите интернет
3. Обновите страницу — работает из кэша
